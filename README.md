# E-Plant Shopping

## Description
E-Plant Shopping is a React-based web application that allows users to browse plants and manage a shopping cart. Users can add items to the cart, view selected items, and manage quantities.

## Features
- Display list of plant products
- Add products to cart
- Remove products from cart
- Update item quantities
- View total price
- Responsive user interface

## Technologies Used
- React
- Redux Toolkit
- JavaScript (ES6)
- HTML5
- CSS3

## Installation

1. Clone the repository:
   git clone https://github.com/achuakshay846-prog/e-plantShopping

2. Navigate to the project folder:
   cd e-plantShopping

3. Install dependencies:
   npm install

4. Run the application:
   npm run dev

## Project Structure
- App.jsx – Main application component
- ProductList.jsx – Displays plant products
- CartItem.jsx – Individual cart item component
- CartSlice.jsx – Redux slice for cart logic
- store.js – Redux store configuration
